# Support

Housing Application Packet Builder is an unreleased, synthetic-only demonstration maintained on a best-effort basis. There is no production support commitment, service-level agreement, or authorization to process real applicant information.

## Before requesting help

1. Read [README.md](./README.md) and [docs/local-development.md](./docs/local-development.md).
2. Confirm Node.js 22 or newer, `DATA_MODE=synthetic`, and the mock processor.
3. Run the smallest relevant verification command.
4. Search existing issues for the same sanitized error.

## Public support requests

Use a repository issue for reproducible bugs or narrowly scoped feature proposals. Include the revision, operating system, Node/npm versions, command, sanitized error, and synthetic reproduction steps.

Never attach real records, uploaded document contents, database files, filled environment files, provider keys, session values, or sensitive logs. Remove identifiers from screenshots and logs before sharing them.

## Security reports

Do not use a public issue for a vulnerability or exposed secret. Follow [SECURITY.md](./SECURITY.md).

## Scope

Maintainers may help with the documented local Node/npm and SQLite workflow. Production deployment, real-data authorization, regulatory interpretation, agency integration, and vendor-contract review are outside Phase 0 support.
